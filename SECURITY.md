# Security Policy

## Supported versions

The latest published version of `@joseftmson/notion-sync` is the supported line.

## Reporting a vulnerability

Please do not open public GitHub issues for security-sensitive findings.

Report security issues privately with:
- a short description
- affected command or flow
- impact
- reproduction steps
- whether secrets or local data exposure is involved

Until a private contact channel is added, do not publish proof-of-concept secrets or live tokens in issues, pull requests, or discussions.

## Scope

Relevant areas include:
- Notion credential handling
- secret masking failures
- local export leakage
- remote upload exposure
- unsafe logging of sensitive data
